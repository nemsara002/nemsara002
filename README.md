### 🤖 Featured Projects

**[Parkinson's Companion Robot](https://github.com/nemsara002/parkinsons-companion-robot)** — *Individual Design Project*
A semi-autonomous assistive robot that follows a Parkinson's patient around the home and
detects falls from a camera feed. Built on a Raspberry Pi 4 + ESP32 differential-drive platform.
- Vision-based **fall detection** pipeline (MediaPipe pose → handcrafted temporal features → Random Forest classifier)
- **Person-following** controller outputting differential-drive signals to an ESP32
- Live deployment scripts for on-device (Raspberry Pi) operation
- *Tools: Python, OpenCV, MediaPipe, scikit-learn, Raspberry Pi, ESP32*

**[Control Systems — Line-Follower Digital Twin](https://github.com/nemsara002/control-systems-line-follower)** — *Team Project (MC3113)*
Control systems engineer on a team line-follower digital twin. Evolved the controller from a
basic fixed-gain PID into a **multi-mode adaptive controller** handling line tracking, obstacle
avoidance, and fault recovery.
- PID tuning, **anti-windup**, filtered/limited derivative, steering saturation
- Scenario-aware **state machine** (tracking / obstacle / fault modes) with state resets
- *Tools: MATLAB, Simulink, control systems design*
